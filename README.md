# 🌾 Projeto Agro Sustentável - Agrinho 2026

Este projeto foi desenvolvido para participar do concurso **Agrinho 2026**, na categoria de programação/desenvolvimento web. O site tem como objetivo conscientizar produtores e a comunidade sobre a importância de práticas agrícolas sustentáveis.

## 🚀 Funcionalidades e Diferenciais
- **HTML Semântico:** Uso correto das tags estruturais para melhor acessibilidade e SEO.
- **Layout Responsivo:** Visual otimizado para celulares, tablets e computadores usando CSS Grid e Flexbox.
- **Interatividade (JS):** Simulador inteligente que calcula a economia diária de água ao adotar métodos eficientes de irrigação.

## 🛠️ Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript (Vanilla)

## 📌 Como Executar o Projeto
1. Baixe a pasta do projeto.
2. Abra o arquivo `index.html` em qualquer navegador web de sua preferência.
